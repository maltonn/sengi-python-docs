## 数値と数字
コンピュータ上では数値と数字が明確に区別されます。下の例を見て下さい。
```python
print(1+2) 
print("1+2") 
```
出力は下のようになります。
```python
3
1+2
```
上では `1+2` が数値として扱われているので、計算した結果が出力されます。一方で、下では `1+2` が数字、すなわち文字として扱われているので、`1+2` という文字がそのまま出力されます。

一般に `"` (ダブルクォーテーション)で囲まれた部分は文字として扱われます。

## 変数
ということで、変数というものを使ってみましょう。変数とは、数値や文字の値が保存されている場所に名前を付けたものです。名前を付けることによって、中身の値が変わっても同じものとして扱えます。

実際の例を通じて理解を深めましょう。下は縦5横8の長方形の面積を出力するプログラムです。
```python
tate=5
yoko=8
print(tayo*yoko)
```
出力は下のようになります。
```python
40
```
ここでは `tate` , `yoko` が変数です。

## 代入
変数に値をぶち込むことを代入と言います。`=` は右の値を左の変数に代入することを意味します。数学の`=` とは意味が異なるので注意しましょう。
さきほどの例では `tate=5` や `yoko=8` が代入を意味します。

数学との違いを理解するために、下の2つの例を見てみましょう。

### 例1
```python
a+1=2
print(a)
```
これはエラーで実行されません。なぜなら`=` の左が変数ではないからです。

### 例2
```python
a=1
a=a+2
print(a)
```
出力は下のようになります。
```python
3
```
出力結果に納得できますか？おそらく2行目の解釈に戸惑っているのではないでしょうか？このプログラムは次のように実行されます。

① 変数aに1を代入する
↓
② 変数aにa+2を代入する（つまりaに1+2すなわち3が代入される）
↓
③ 変数aの中身の3を出力する

## 条件式とは
値や式の比較や論理演算を組み合わせたもので、計算結果は条件式が正しいときはTrue(真)、正しくないときはFalse(偽)になります。

## 条件を表す記号

### 比較演算子
 == 右と左が等しい
\>= 左が右以上
 <= 左が右以下

### 論理演算子
not ～でない
and かつ
or または

# 6.条件分岐

## if文
条件によって処理の内容を変えることを条件分岐といいます。条件分岐を行うにはif文というものを使います。下のように書きます。
```
a=int(input())
if (a>=10):
    print(a+"は10以上です")
```
`if` は「もし～ならば」を意味します。()内は前章で扱った条件式です。`if (条件式):`で「もし条件式がTrueならば」を意味します。ここでは「もしaが10以上ならば」ということになります。

条件を満たしているときの処理は、`if` に対してスペースを4つ入れることで指定します。Tabキーを押すとスペースが4つ入ります。スペースの数を間違えるとエラーになるので注意してください。

## else
条件を満たさないときの処理も指定するときは `else` を使います。下のように書きます。
```
a=int(input())
if (a>=10):
    print(a+"は10以上です")
else:
    print(a+"は10未満です")
```
`else` は「もしそうでなければ」を意味します。

`else` は下のようにIf文で代用することもできますが、条件が同時に起きる場合があるなどのミスの原因になるので、分岐が2つであればやめておくのが無難でしょう。
```
a=int(input())
if (a>=10):
    print(a+"は10以上です")
if (a<10):
    print(a+"は10未満です")
```
## elif
3つ以上に条件分岐したいときは、`elif` を使います。下のように書きます。
```
a=int(input())
if (a>=18):
    print("大人料金です")
elif (a>=12):
    print("中高生料金です")
else:
    print("子供料金です")
```
`elif` は「もしそうでなかったとき、もし～ならば」を意味します。ここでは「もしaが18以上でなかったとき、もしaが12以上ならば」ということになります。

こちらも if文 で代用することができます。
```
a=int(input())
if (18<=a):
    print("大人料金です")
if (12<=a<18):
    print("中高生料金です")
    
その話でー下の方にやばしが書いたIf文を重ねる時の書き方講座(コラム)があるんやけど、読んでください
if (a<12):
    print("子供料金です")
```
こちらはたぶん許容されます。でも条件が重複していないかのチェックは必要ですね。たぶん。

こういうときに、条件式の大小関係を揃えるというような配慮ができるとわかりやすいプログラムになります。たぶん。

# for文
同じ処理を繰り返したり、配列の中身を順番に取り出したい時はfor文を使います。
ただしforもifと同じように実行する範囲を指定するためにスペースを入れる必要があります。

> [name=Suburi Soburi][color=#f9a909]「繰り返す対象となる処理をインデントすることで指定します」のほうが適切では

まずは同じ処理を繰り返す使い方です
```python
for i in range(5):
    print("Hello World")#forの中身はここまで
print("end")#ここはforの中身に含まれない
```
これを実行すると出力は以下のようになります。
forの範囲内がrangeの中に書かれた数字と同じ回数ループするのがわかります。
```
Hello World
Hello World
Hello World
Hello World
Hello World
end
```

次に配列の中身を順番に取り出すという使い方です

```python
L=[1,3,5,7]
for i in L:
    print(i) 
    #forの中身はここまで
#ここはforの中身に含まれない
```
上のプログラムを実行すると出力は以下のようになります。
```
1
3
5
7
```
最初はLの中の0番目の要素である1が変数iに入った状態でforの中身が実行されます。
そして一通り実行した後は変数iに1番目の3が入った状態でもう一度forの中身が実行されます。
これが繰り返されることで結果的に配列の中身がすべて表示されます


ちなみにrangeでループする理由ですが
range(n)は[0,1,2,・・・,n-1]と同じ意味だと思って貰えばいいです
```python
for i in range(5):
    print(i)
```
とすれば下のようになります
```
0
1
2
3
4
```

また繰り返しを途中で中断したい場合はbreakを使います
```python
L=[1,6,3,7,5,2,4]
for i in L:
    print(i)
    if(i>6):
        break
```
これを実行すると以下のようになります
i=７になったときにi>6を満たして繰り返しが終了します
```
1
6
3
7
```

# はじめに
プログラミングというものの立ち位置を説明します。

## 機械語
機械語とは、コンピュータが直接解釈可能な言語です。0と1だけを用いて記述されています。

## プログラム
プログラムとは、コンピュータに対する命令を記述したものです。機械語で記述されます。

## プログラミング言語
機械語は0と1しか用いないので、常人には理解できず書けません。そこでプログラミング言語の登場です。プログラミング言語は、機械語より理解しやすく、かつ機械語にちゃんと翻訳できるように作られた言語です。C言語, C++, Java, JavaScript などいろいろな種類があります。

### 分類

#### コンパイラ方式とインタプリタ方式

#### 手続き指向とオブジェクト指向

## ソースコード
プログラミング言語で書かれたものをソースコードと言います。


# ソースコードの単位
日本語の文章には言葉の単位があります。段落、文、文節、単語、文字などです。
ソースコードにも言葉の単位があります。ブロック、文、トークンです。

## ブロック
ブロックとはいくつかの文を { と } で囲んだものです。ひとつのまとまりとして認識されます。

## 文
文は実行する動作を記述したものです。いくつかのトークンで構成されていて、;(セミコロン)で終わります。

## トークン
トークンはソースコードの最小単位です。品詞みたいなものです。トークンは空白類（スペース,タブ,改行）で分ける必要があります。こちらは句読点みたいなものです。トークンには主に以下の6つがあります。

### 識別子
操作するデータや命令を識別するためにソース上で命名された名前です。変数名や関数名などがこれにあたります。

### 予約語
予約語とは、プログラミング言語上で特定の役割が与えられている単語です。制御構造やデータ型を表す単語などがこれにあたります。

### 定数
定数とは、ソースコード上で指定された固定的な値です。定数には、数値だけでなく文字列も含まれます。10 や 3.14 や "Hello, World!" などがこれにあたります。

### 演算子
演算子は、計算のために用いる記号です。+ - * % = == < > && などがこれにあたります。

### 区切り子
区切り子とは、区切りやまとまりを表すために用いる記号です。[ ] ( ) { } , : ; などがこれにあたります。

### コメント
コメントとはプログラムの処理と一切関係のない単なるメモです。 他人の書いたコードを理解することは難しいので、適度な注釈はコードの理解を高めます。
C言語では2種類の方法があり、"//"から行の終わりまで、または"/*"と"*/"とで囲まれた部分がコメントになります。

# 関数

## 性質
関数は、いくつかの処理をまとめたものです。関数を用いることによって、同じ処理をまとめて書くことができ、以後その関数を呼び出すだけでまとめられた処理が実行できます。
関数の処理に必要な値を引数、関数で処理した結果の値を戻り値と言います。引数や戻り値がない場合もあります。

## 種類

### 自作関数
文字通り自分で作った関数です。同じ処理がソースコードのあちこちに出てくるときは、その処理を自作関数にまとめると楽ができます。

### 標準ライブラリ関数
標準ライブラリ関数は、よく用いられる定型的な処理を行う関数を集めた関数です。プログラムの一番初めに下のように記述することで利用できます
```
	#include <stdio.h>
```
stdio.hがライブラリ名です。stdio.hは入出力処理用ライブラリです。そのほかにも数学処理などのライブラリが存在します。

#### printf関数
入出力処理用ライブラリ<stdio.h>に含まれている出力用の関数です。定数(文字列を含む)を出力するときは下のように記述します。
```
	printf("定数");
```
変数の値を出力するときは下のように記述します。
```
	printf("%d",変数名);
```
#### scanf関数
入出力処理用ライブラリ<stdio.h>に含まれている入力用の関数です。下のように記述します。
```
    int 変数名;
	scanf("%d",&変数名);
```
printf関数と異なり、変数名の前に"&"が必要であることに注意してください。

## 扱い方

### 定義
関数の定義が必要なのは、関数を自作する場合です。
```
戻り値のデータ型 関数名(引数){
	処理
}
```
例えば下のように定義します。

```
int function(int a, int b) {
	return a+b;
}
```
上の例では、引数はaとb、戻り値はa+bです。return文で戻り値を指定します。また、a+bは整数なので、戻り値のデータ型は整数型のintです。

引数のない関数もあります。その場合は下のように定義します。
```
void HelloWorld(){
	printf("Hello, World!");
}
```
引数がないときは、本来引数を記述する部分に何も記述しなければよいわけです。

### 呼び出し
戻り値のある関数の場合は下のように呼び出します。
```
#include <stdio.h>
int function(int a, int b) {
	return a+b;
}
int main(void){
	int x;
	x=function(1,2);//関数の呼び出し
}
```
このように戻り値は定数のように扱えます。

引数や戻り値のない関数の場合は下のように呼び出します。
```
#include <stdio.h>
void HelloWorld(){
	printf("Hello, World!");
}
int main(void){
	HelloWorld();//関数の呼び出し
}
```
引数がないときは本来引数を記述する部分に何も記述しなければよく、戻り値がないときは単に"関数名();"と記述すればよいわけです。

標準ライブラリ関数のprintf関数は、引数があり戻り値がないので、下のように記述します。
```
#include <stdio.h>
int main(void){
	printf("文章");
}
```
（なんかprintf関数って戻り値（出力文字数）存在するみたいだね、使わないだけで、まぁいっか）
